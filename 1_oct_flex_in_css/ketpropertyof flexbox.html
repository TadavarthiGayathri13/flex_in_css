Defines how the browser distributes space between and around items along the main axis of the container.
Values:
flex-start: Items are packed toward the start of the main axis.
flex-end: Items are packed toward the end of the main axis.
center: Items are centered along the main axis.
space-between: Items are evenly distributed with the first item at the start and the last item at the end.
space-around: Items are evenly distributed with equal space around them.
space-evenly: Items are evenly distributed with equal space around them, including at the start and end.
 Defines how the browser aligns items along the cross axis of the container.
Values:
flex-start:Items are aligned at the start of the cross axis.
flex-end: Items are aligned at the end of the cross axis.
center: Items are centered along the cross axis.
baseline: Items are aligned such that their baselines align.
stretch: Items are stretched to fill the container along the cross axis.
 Sets the size of the gap between items in a flex container.
Values:
A length value (e.g., 10px, 2em).
A percentage value (e.g., 5%).
normal: The default value, which is browser-specific and typically equivalent to 1em.
 Establishes the main axis along which flex items are placed in the flex container.
Values:
row: Items are placed along the main axis (horizontal) in the order they appear in the source code.
row-reverse: Items are placed in the opposite order of the source code.
column: Items are placed along the cross axis (vertical).
column-reverse: Items are placed in the opposite order along the cross axis.
 Specifies whether flex items are forced onto a single line or can be wrapped onto multiple lines.
Values:
nowrap: All flex items are on one line (the default behavior).
wrap: Flex items wrap onto multiple lines if needed.
wrap-reverse:Flex items wrap onto multiple lines in the reverse order.
